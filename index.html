ejercicio1.1

Solicitar un dato por pantalla (prompt) y mostrarlo en un párrafo.

Aquí aparecerá el texto introducido.

ejercicio1.2

Seleccionar de una lista plegada un color y mostrarlo por la consola.

Abre la consola del navegador (F12) para ver el color seleccionado.

ejercicio1.3

Solicitar en un input un texto y mostrarlo en una capa <div> pulsando sobre un botón.

El texto aparecerá aquí al pulsar Mostrar.

ejercicio1.4

Cargar una lista plegable con nombres de imágenes locales y visualizar la imagen seleccionada.

Vista previa de la imagen seleccionada.

ejercicio1.5

Crear una página con un texto y dos listas plegables: una para negrita y otra para cursiva. Aplicar al texto mostrado.

Aquí se mostrará el texto con los estilos seleccionados.
